
# 🚀 FUTURE INTERNS - MACHINE LEARNING TASK 1

## 📊 Project Title
Sales & Demand Forecasting for Businesses

## 🎯 Objective
The goal of this project is to **predict future sales using historical sales data**.

Sales forecasting helps businesses:

- 📦 Plan inventory  
- 💰 Manage cash flow  
- 👥 Prepare staffing  
- ⚠️ Avoid overstocking or losses  

---

- 🐍 Python  
- 🐼 Pandas  
- 🔢 NumPy  
- 🤖 Scikit-learn  
- 📈 Matplotlib  
- 📓 Jupyter Notebook  

---

## 🤖 Machine Learning Model
Linear Regression

3️⃣ Time feature extraction  
4️⃣ Data visualization

6️⃣ Model evaluation  
7️⃣ Future sales prediction  

---

## 📈 Result
The model successfully predicts **future sales trends based on historical data**, helping businesses make better decisions.
